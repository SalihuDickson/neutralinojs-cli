The official CLI of Neutralinojs. See neu CLI [documentation](https://neutralino.js.org/docs/cli/neu-cli/) for more details.

```bash
npm i -g @neutralinojs/neu
```
### Requirements

- Node.js v14 or above.
- npm or yarn package manager.
